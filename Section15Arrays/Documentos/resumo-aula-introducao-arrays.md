# Resumo - Introdução a Arrays em Java

Array é um objeto em Java usado para armazenar vários valores do mesmo tipo.

Enquanto uma variável comum guarda apenas um valor, um array permite guardar vários valores dentro de uma única estrutura.

Exemplo de variável comum:

int idade = 38;

Exemplo de array:

int[] idades = new int[5];

Arrays ajudam a evitar a criação de várias variáveis repetidas, como idade1, idade2, idade3 etc.

Apesar de Collections serem mais modernas, flexíveis e muito usadas em projetos reais, arrays continuam sendo importantes para entender a base da linguagem Java, dar manutenção em códigos antigos e preparar o raciocínio para estruturas mais avançadas.

Neste capítulo, o foco será entender arrays com calma antes de avançar para Collections.
