# Resumo Final - Seção 15 - Arrays

Nesta seção estudei os fundamentos de arrays em Java.

## Conceitos praticados

- Declaração de arrays
- Criação de arrays com tamanho fixo
- Preenchimento manual de posições
- Acesso por índice
- Uso de length
- Percurso com for tradicional
- Percurso com for-each
- Alteração de elementos
- Diferença entre tamanho do array e último índice

## Erros e aprendizados

Um erro importante foi tentar percorrer um array de double usando uma variável int no for-each. Aprendi que o tipo da variável usada no for-each precisa ser compatível com o tipo do array.

Também fixei que array.length retorna o tamanho do array, mas o último elemento é acessado com:

array[array.length - 1]

## Relação com Java back-end

Arrays ajudam a entender como trabalhar com vários valores em uma única estrutura. Embora em projetos modernos Java seja mais comum usar Collections, entender arrays é importante para base da linguagem, manutenção de código e preparação para listas, coleções e manipulação de dados em sistemas reais.
