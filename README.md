# JavaModerno – Diário de Estudos em Java

## 📌 Introdução

Neste repositório registro minha jornada de aprendizado em Java ao longo do curso Java Moderno do prof° Arnaldo Sousa.  
O objetivo é:

- Consolidar o conteúdo aprendido em cada seção/módulo.
- Manter um histórico de exercícios e pequenos projetos.
- Criar um portfólio público que mostre minha evolução como desenvolvedor Java.

---

## 📚 Conteúdo Aprendido

Nesta seção, resumo os principais tópicos estudados em cada parte do curso.

### Seção 10 – Fundamentos da Programação em Java
- Conceitos básicos: `class`, `main`, `System.out.println`
- Declaração de variáveis e tipos primitivos (`int`, `double`, `boolean`, `char`)
- Inferência de tipos com `var`
- Promoções automáticas de tipo

### Seção 11 – Operadores
- Operadores aritméticos (`+`, `-`, `*`, `/`, `%`)
- Operadores relacionais (`>`, `<`, `>=`, `<=`, `==`, `!=`)
- Operadores lógicos (`&&`, `||`, `!`)
- Precedência de operadores

### Seção 12 – Controles de Fluxo
- `if`, `else if`, `else`
- `switch`
- Uso de `Scanner` para entrada de dados

### Seção 13 – Loops
- `for`, `while`, `do-while`
- Uso de `break` e `continue`
- Exercícios práticos com repetições

### Seção 14 – Princípios de Programação Orientada a Objetos (POO)
- Classes e objetos
- Variáveis de instância (atributos)
- Construtores (com e sem parâmetros)
- Sobrecarga de construtores
- Palavra-chave `this`

> À medida que eu avançar no curso, vou adicionar novas seções e tópicos aqui.
